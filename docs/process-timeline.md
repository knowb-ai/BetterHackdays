# Process Timeline

## Purpose

Provide a simple execution plan from event discovery to demo day.

## Example stages

- before the event
- first 30 minutes
- first 2 hours
- same day validation
- demo preparation
- final submission

## What the timeline should include

- key tasks
- deadlines
- decision checkpoints
- risk flags
- optional help prompts
